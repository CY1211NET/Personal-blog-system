# 博客系统项目介绍

## 📖 项目概述

这是一个功能完善的**全栈博客系统**，采用现代化的技术栈构建，提供了从文章创作、发布、浏览到互动的完整博客体验。系统设计为**单用户博客**，适合个人博客、技术文档站点等场景。

### 核心特性

- ✨ **现代化界面设计** - 采用暗黑科幻风格，提供优雅的用户体验
- 📝 **Markdown 编辑器** - 集成 Vditor 编辑器，支持实时预览和丰富的 Markdown 语法
- 🎨 **代码高亮** - 使用 highlight.js 提供多语言代码高亮显示
- 🏷️ **分类与标签** - 灵活的文章分类和标签系统
- 💬 **评论系统** - 支持文章评论和互动
- 👍 **点赞功能** - 文章点赞和阅读量统计
- 🔍 **搜索功能** - 支持按标题和内容搜索文章
- 📱 **响应式设计** - 适配各种屏幕尺寸
- 🌐 **国际化支持** - 内置中英文切换
- 🔐 **JWT 身份认证** - 安全的用户认证机制
- 📊 **时间线视图** - 以时间轴形式展示文章历史
- 👤 **个人资料管理** - 支持头像上传、个人简介、社交链接等

## 🎯 核心功能详解

### 1. 用户系统

#### 单用户设计
- 系统设计为单用户博客，首次注册后将禁止其他用户注册
- 提供注册状态检查接口，前端可根据状态控制注册入口显示

#### 身份认证
- 基于 JWT Token 的无状态认证
- Token 有效期可配置（默认 24 小时）
- 自动刷新机制，保持登录状态

#### 个人资料管理
- 头像上传和更新
- 个人简介编辑
- 社交链接管理（GitHub、Twitter、Email 等）
- 个人信息展示

### 2. 文章系统

#### 文章创作
- **Vditor 编辑器**：功能强大的 Markdown 编辑器
  - 实时预览
  - 工具栏快捷操作
  - 支持表格、代码块、数学公式等
- **分类管理**：支持创建和选择文章分类
- **标签系统**：多标签支持，逗号分隔
- **草稿保存**：自动保存编辑内容

#### 文章展示
- **列表视图**：
  - 分页展示
  - 文章摘要
  - 分类和标签显示
  - 阅读量和点赞数统计
- **详情页面**：
  - Markdown 渲染
  - 代码高亮
  - 目录导航（TOC）
  - 代码块复制功能
  - 表格折叠功能
- **时间线视图**：
  - 按时间轴展示文章
  - 直观的历史记录

#### 文章管理
- 编辑已发布文章
- 删除文章（仅作者可操作）
- 文章搜索（标题和内容）
- 按分类筛选
- 按标签筛选

### 3. 互动功能

#### 评论系统
- 发表评论
- 评论列表展示
- 评论时间显示
- 评论者信息展示

#### 点赞功能
- 文章点赞
- 点赞数统计
- 防重复点赞

#### 阅读统计
- 自动记录阅读量
- 阅读数展示

### 4. 分类与标签

#### 分类管理
- 创建新分类
- 分类列表展示
- 按分类筛选文章
- 分类文章数统计

#### 标签系统
- 标签云展示
- 标签权重显示
- 按标签筛选文章
- 热门标签推荐

### 5. 搜索功能
- 全文搜索
- 标题搜索
- 内容搜索
- 实时搜索结果

### 6. 文件上传
- 图片上传
- 头像上传
- 文件大小限制
- 文件类型验证
- 安全的文件存储

### 后端安全
- **JWT 认证**：安全的 Token 机制
- **密码加密**：使用 bcrypt 加密存储
- **CORS 配置**：跨域请求控制
- **SQL 注入防护**：GORM 参数化查询
- **文件上传验证**：类型和大小限制

### 前端安全
- **XSS 防护**：使用 DOMPurify 清理 HTML
- **路由守卫**：未授权访问拦截
- **Token 管理**：安全的 Token 存储
- **输入验证**：表单数据验证

## 🚀 部署说明

详细的部署步骤请参考 [README.md](./README.md) 文件。

### 快速开始

**后端：**
```bash
cd backend
go run main.go
```

**前端：**
```bash
cd frontend
npm install
npm run dev
```

### 生产部署

**后端：**
```bash
cd backend
go build -o blog-backend main.go
./blog-backend
```

**前端：**
```bash
cd frontend
npm run build
# 将 dist 目录部署到 Nginx 或其他 Web 服务器
```
